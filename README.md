# 📚 Study Planner with Priority System

---

## 🧠 Overview

The **Study Planner with Priority System** is a simple command-line Python application designed to help students manage their academic tasks efficiently. It allows users to add tasks, assign priorities, track completion, and stay organized.

---

## ❗ Problem Statement

Students often:

* Forget important tasks
* Struggle with prioritization
* Have poor time management

This project provides a simple solution to organize study tasks effectively.

---

## 💡 Solution

This application allows users to:

* Add tasks with priority (High, Medium, Low)
* View tasks sorted by priority
* Mark tasks as completed
* Delete tasks
* Save tasks using JSON

---

## 🚀 Features

* ✅ Add tasks
* 🔺 Priority-based sorting
* ✔ Mark tasks complete
* 🗑 Delete tasks
* 💾 Persistent storage
